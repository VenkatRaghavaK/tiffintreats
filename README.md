# TiffinTreats 

## Overview  
TiffinTreats is a simple, responsive web-app for ordering home-style South Indian tiffins.  
- Users can browse a menu, search items, adjust quantities, view a cart summary, and checkout via Razorpay.  
- Built with **HTML, CSS, and vanilla JavaScript** — no framework required, ensuring ease of maintenance and portability.

## Use Case  
- Ideal for small food businesses or home-based kitchens.  
- Demonstrates core e-commerce flows: menu listing, dynamic cart, real-time price/quantity update, and payment integration.  
- Works well on desktop, tablet, and mobile — responsive design included.

## Features ✅  
- Responsive UI (desktop → mobile) with flexible layout and styling.  
- Search functionality — filter dishes by name as you type.  
- Add / remove item quantity per dish.  
- Real-time cart summary & total price update.  
- Cart popup to review order before checkout.  
- Razorpay payment integration for checkout.
